# Tipos de dados

Conforme o EMAScript standard temos 9 tipos de dados:

* Data types
    * Primitive / Primitive value
    * Structal
    * Strunctural Primitive

 ## Primitivos

 * String   "ABC" "BCA "bca" "Bca" "bCa"
 * Number
 * Boolean
 * Underfined
 * Symbol
 * BigINT   

 ## Estruturais

 * Object
    * Array
    * Map
    * Set
    * Date
    * ...
* Function
